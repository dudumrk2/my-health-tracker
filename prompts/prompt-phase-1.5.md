## שינוי 1 — Gender בפרופיל

**מה לשנות:**

1. **מודל הנתונים** — הוסף שדה `gender: String` (`"male"` | `"female"` | `"other"`) למחלקת הפרופיל (data class / model).

2. **Firestore** — עדכן את ה-repository כך שישמור ויקרא את `gender` תחת `users/{uid}/profile`.

3. **מסך פרופיל** — הוסף בחירת מין (3 אפשרויות: Male / Female / Other) למסך הפרופיל הקיים. ולידציה: שדה חובה.

4. **העברה ל-AI (לעתיד)** — `gender` כבר צריך להיות זמין ב-ViewModel/Repository כדי שפייז 2 ו-3 יוכלו לשלוח אותו ל-Gemini כהקשר. אין צורך לממש את ההעברה עצמה כאן — רק לוודא שהשדה נשמר ונקרא.

## שינוי 2 — אימון ידני

**מה לשנות:**

1. **מודל הנתונים** — הוסף שדה `source: String` (`"health_connect"` | `"manual"`) לכל אובייקט workout ב-`healthDaily`. עדכן את ה-data class בהתאם. אימונים קיימים מ-Health Connect מקבלים `source = "health_connect"`.

2. **מסך / דיאלוג Add Workout** — הוסף כפתור "+" במסך הבריאות / Dashboard שפותח דיאלוג/מסך לבחירת:
   - סוג אימון (רשימה: ריצה, הליכה, רכיבה, שחייה, כוח, יוגה, אחר — או חיפוש חופשי)
   - משך (דקות)
   - תאריך ושעת התחלה (ברירת מחדל: עכשיו)

3. **שמירה ל-Firestore** — אימון ידני נשמר ל-`healthDaily/{date}` עם `source: "manual"`. אם ביום יש גם Health Connect וגם ידני — `healthDaily.source = "mixed"`. כתיבה idempotent (אותו מנגנון קיים).

4. **תצוגה ב-Dashboard** — אימונים ידניים מופיעים ברשימת האימונים של היום לצד אימוני Health Connect, עם אינדיקציה ויזואלית קטנה (למשל אייקון עט / תווית "manual").
